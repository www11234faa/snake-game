# 🌐 公网访问部署指南

## 方案1：GitHub Pages（推荐）

### 步骤：
1. 访问 https://github.com
2. 创建新仓库，命名为 `snake-game`
3. 上传以下文件：
   - index.html
   - game.js
   - style.css
4. 在仓库设置中启用GitHub Pages
5. 获得地址：`https://[您的用户名].github.io/snake-game/`

## 方案2：Vercel（最简单）

### 步骤：
1. 访问 https://vercel.com
2. 拖拽整个文件夹到网页
3. 自动部署完成
4. 获得公网地址

## 方案3：Netlify

### 步骤：
1. 访问 https://netlify.com
2. 拖拽文件夹到部署区域
3. 获得公网URL
